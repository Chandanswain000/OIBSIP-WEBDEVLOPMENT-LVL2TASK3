
# OIBSIP_WebDevelopment_Task3

## Interactive To-Do List Web Application

### Objective

The objective of this project is to develop a simple and interactive **To-Do List Web Application** that helps users manage their daily tasks. The application allows users to add tasks, mark them as completed, edit tasks, and delete tasks easily.

### Tools Used

* HTML
* CSS
* JavaScript
* VS Code
* GitHub

### Steps Performed

1. Created the basic structure of the web page using **HTML**.
2. Designed a simple and clean interface using **CSS**.
3. Used **JavaScript** to add functionality for adding tasks dynamically.
4. Implemented **Pending Tasks and Completed Tasks sections** to organize tasks.
5. Added a **checkbox feature** so users can mark tasks as completed.
6. Added the option to **edit tasks by double clicking**.
7. Implemented a **delete button** to remove tasks.
8. Used **LocalStorage** to store tasks so they remain saved after refreshing the page.
9. Uploaded the project to **GitHub repository** for submission.

### Features

* Add new tasks
* View tasks in **Pending Tasks**
* Mark tasks as **Completed**
* View tasks in **Completed Tasks section**
* Edit tasks
* Delete tasks
* Save tasks using **LocalStorage**

### Outcome

Through this project, I learned how to build an interactive web application using **HTML, CSS, and JavaScript**. I also gained practical knowledge of **DOM manipulation, event handling, and browser LocalStorage** to manage and store tasks.
